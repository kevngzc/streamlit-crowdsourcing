How to Use the Portal

Follow these steps to use the Museum Data Crowdsourcing Portal:

  1. Enter your unique token on the login page.
  2. Once logged in, view the data assigned to your country.
  3. Edit the data directly in the table provided.
  4. Save your changes using the "Save Changes" button.

Administrator Access

If you are an administrator, you can:

  • View and edit all available data.
  • Upload new datasets or append to the existing dataset.
  • Generate and manage tokens for participants.

Contact

If you encounter issues or need assistance, please contact the administrator at support@example.com.
